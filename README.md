# Python File & API Practice

This project demonstrates basic Python skills, including file handling, API integration, exception handling, and logging.

1. Features
   
   File Read & Write
   
   Create, append, and read text files (python_notes.txt)
   
   Search for keywords in the file
   
3. API Integration
   
   Fetch products from a dummy API (https://dummyjson.com)
   
   Filter and sort products by rating and price
   
   Fetch laptops category
   
   Add a new product via API
   
5. Exception Handling
   
   Safe division function
   
   Safe file reading
   
   Product lookup with input validation
   
7. Logging
   
   Errors logged to error_log.txt with timestamps
   
   Handles connection errors, timeouts, 404s, and other exceptions
